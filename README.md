# 🚀 HireFlow

A modern AI-assisted job board built with Next.js 15, TypeScript, and a scalable feature-based architecture.

---

## 📌 Overview

HireFlow is a production-style job board designed as a technical assessment project.  
It focuses on clean architecture, reusable UI components, and a premium SaaS-like user experience.

---

## ✨ Features

- Modern landing page with SaaS UI design
- Featured Jobs section
- Featured Companies section
- Category-based job browsing
- How It Works flow
- CTA conversion section
- Fully responsive design (mobile-first)
- Reusable component architecture
- Mock-data driven UI (backend-ready)

---

## 🧱 Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

---

## 📁 Architecture

Feature-first structure:
src/
